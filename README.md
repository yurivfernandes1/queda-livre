# queda-livre
![.NET 8](https://img.shields.io/badge/.NET-8.0-512BD4?logo=dotnet&logoColor=white)

Projeto acadêmico em C# que implementa um sistema de filas para uma companhia aérea.

Resumo
- **Objetivo:** Gerenciar o cadastro de passageiros para 3 voos e uma lista de espera.
- **Funcionalidades:** cadastrar passageiros, exibir listas, pesquisar por nome e CPF e gerenciar a fila de espera.

Pré-requisitos
- .NET SDK 8.0 ou superior (https://dotnet.microsoft.com/)

Como executar
1. Restaurar dependências e compilar:

```bash
dotnet build
```

2. Executar o projeto:

```bash
dotnet run --project .
```

Estrutura principal
- `Program.cs` — ponto de entrada do aplicativo
- `queda-livre.csproj` — arquivo de projeto

Observações
- Este repositório é um projeto acadêmico — desenvolvido para fins de avaliação e aprendizado.

Contribuições
- Este projeto não é mantido ativamente; sinta-se à vontade para abrir issues ou pull requests com melhorias.
